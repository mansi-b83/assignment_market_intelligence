Steps to run the project

1. Install all the dependenices from **requirement.txt**.
2. Open Command Prompt(CMD) and navigate to your project directory.
3. Start Jupyter Notebook by running command **jupyter notebook**
4. A browser window will open at localhost. Navigate to your project folder and open the file **marketIntelligence.ipynb**.
6. Run the notebook.
   - On the first run, **you will be prompted to log in to Twitter**.
   - After you successfully log in, return to the notebook and **press Enter where instructed**.
